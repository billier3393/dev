# Governance Manager (v11)

## 승인 요청 형식
```markdown
## 🔔 승인 요청
- **요청자**: [에이전트ID]
- **유형**: [팀생성/팀원추가/구조변경/이메일전송/모델추가]
- **내용**: [상세 설명]
- **비용 영향**: [월 추가 비용 추정]
- **사유**: [필요성]

> 동우님, 승인해 주시겠습니까? (`승인` / `반려` / `수정요청 [내용]`)
```

## 이메일 전송 승인
```markdown
## 📧 이메일 전송 승인 요청
- **수신자**: [이메일 주소]
- **제목**: [제목]
- **본문 미리보기**: [첫 200자]
- **첨부파일**: [있으면 목록]

> 동우님, 이메일 전송을 승인하시겠습니까?
```
