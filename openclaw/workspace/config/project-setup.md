# 프로젝트 디렉토리 초기 설정

```bash
# 저장소 루트에서 실행
cd /path/to/openclaw/workspace
bash scripts/team-init.sh
```

## 사용법
1. `팀 시작` — 전체 초기화
2. `연구 [주제]` — 리서치 체인
3. `개발 [작업]` — 개발 체인
4. `분석 [데이터]` — 분석 체인
5. `보고서 [주제]` — 보고서 체인
6. `기획 [아이디어]` — 크리에이티브 체인
7. `보안점검` — 보안 체인
8. `최적화 [대상]` — 최적화 체인
9. `배치 [작업]` — Batch API 체인 (v11 신규)
10. `스킬생성 [이름]` — Skill Factory 체인 (v11 신규)
11. `예산 현황` — 일일/월간 비용 현황 (v11 신규)
12. `메시지 현황` — 에이전트 메시지 큐 상태 (v11 신규)
13. `Lock 현황` — 공유 리소스 Lock 상태 (v11 신규)

## 레거시 (v9 호환)
- `OCR [파일명]` → OCR 체인
- `공부 [챕터]` → 리서치 체인
- `코드 [작업]` → 개발 체인
